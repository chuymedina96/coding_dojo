@media queries

CSS uses the @ media rule to detect information abou the device being used to access the website. For example, it can used to detect the size of the device, its capabilities, and its orientation. For this course, we will be using it to detect the device size, but keep in mind that is it capable of much more.

Desktop: Width greater than or equal to 1024px

Tablet: Width between 481 px and 1023px

Phone: Width less than or equal to 480px
